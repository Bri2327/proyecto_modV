# Vue 3 + Vite
# Proyecto Final Vue 3

## Descripción
Aplicación web desarrollada con Vue 3 para la gestión de productos y categorías.
Incluye autenticación, CRUD completo, filtro, buscador, rutas protegidas y detalle dinámico.

## Tecnologías utilizadas
- Vue 3
- Vue Router
- Axios
- JSON Server
- JSON Server Auth
- CSS

## Instalación
1. Clonar el repositorio
2. Instalar dependencias:

npm install

## Configuración
Crear un archivo `.env` con el siguiente contenido:

VITE_API_URL=http://localhost:3000

## Ejecutar backend
npm run server

## Ejecutar frontend
npm run dev

## Funcionalidades
- Registro de usuario
- Inicio de sesión
- Protección de rutas
- CRUD de categorías
- CRUD de productos
- Filtro de productos por categoría
- Búsqueda de productos por nombre
- Vista de detalle de producto

## Nota
No incluir la carpeta `node_modules` en el repositorio.
El archivo `db.json` debe estar incluido en el proyecto.
